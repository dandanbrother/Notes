#### TCP & UDP：传输层协议

| 特点     | TCP                         | UDP        |
| -------- | --------------------------- | ---------- |
| 连接性   | 面向连接 (三次握手建立连接) | 面向非连接 |
| 可靠性   | 可靠                        | 不可靠     |
| 传输效率 | 慢                          | 快         |

可靠：TCP 接收端会通过校验和对报文检验，成功则返回一个确认报文，否则会重传，根据报文编号是否丢弃重复。

三次握手：1. SYN seq= j; 2. ACK = j+1, SYN = k; 3. ACK = k+1;

四次挥手：1. FIN seq = x, ACK = y; 2. ACK = x+1; 3. FIN seq = y; 4. ACK = y+1;
